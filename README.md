# A Webcrawler written in Golang

Crawls a single domain, printing out a list of assets and links for each new page that it finds.

## Crawl and Scan

Clone the repo into `$GOPATH/src` and do the following:

get deps:

```bash
  # inside the repo
  $ go get
```

build:

```bash
  # inside the repo
  $ go build
```

crawl and scan:

```bash
  python3 <link>
```
